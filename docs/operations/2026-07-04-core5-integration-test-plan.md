## 2. 시나리오

### S1. 메인 파이프라인: 일정 → 대화 → 지식 → 문서 (Q1·Q2·Q3·Q6)
1. 일정 등록("AI혁신 주간보고 준비") → 연결 세션 생성 → 업무대화 자동 전환·연결 표시
2. 대화: "지식폴더에서 AI혁신 추진 현황 근거 찾아 정리해줘" → 젬마 응답 + 출처
3. [문서작성으로 이어가기] → 세션 기반 모드 프리필 확인
4. 1페이지 보고서 → [구조 생성→검토] (단계 스트림 표시) → 위계 편집 → 미리보기 → [이대로 문서 생성] → content base + finalize 안내

### S2. 파일찾기 → 대화 연계 (Q2·Q6)
1. 파일찾기 "빠르쥬" 검색 → 결과 행 [세션에 연결]
2. 업무대화 이동 → 연결 파일 배지 +1 반영
3. 관련 질문 → 응답이 연결 파일 맥락 반영(세션 부스트)

### S3. 지식폴더 단독 (Q2·Q3)
1. 검색 탭 "안전관리등급제" → 트라이그램 결과·품질 배지
2. [근거 답변 생성] → LLM 합성 배지 + 인용 목록
3. 위키 탭 트리(주제/업무 기록/문서) 탐색 → 페이지 열람 → 목차 복귀
4. 대시보드 상태 카드 확인

### S4. 세션 지식 반영 → 업무 기록 (Work-Aware 루프) (Q3·Q6)
1. S1 세션에서 [이 세션 지식 반영]
2. 위키 탭 → "업무 기록"에 세션 페이지 생성, 요약+인용 문서 링크 확인
3. index.md의 업무 기록 섹션 갱신 확인

### S5. 실패/엣지 (Q7)
1. 존재하지 않는 모델명으로 임시 변경 후 대화 → 한국어 실패 안내 → 복원
2. 문서작성 JSON 편집으로 스키마 위반 → 400 힌트 인라인 표시
