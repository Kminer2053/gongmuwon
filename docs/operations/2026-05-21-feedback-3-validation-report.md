## 수정 요약

- 상단 타이틀 설명을 `대화에서 시작해 일정, 검색, 지식, 문서작성, 실행기록까지 한 워크플로로 묶습니다.`로 정리했다.
- 상단 현재 기능 아이콘이 좌상단으로 치우치지 않도록 아이콘 컨테이너 스타일을 조정했다.
- 문서작성 화면을 `대화세션 선택 -> 파일/용도 입력 -> 산출보고서 선택 -> 양식 선택 -> 작업 설명 -> 작업 시작` 흐름으로 단순화했다.
- 문서작성 `작업 시작`에서 승인 단계를 거치지 않고 `/api/documents/generate`로 HWPX 산출물을 생성하도록 추가했다.
- 업무대화에서 `hwpx`, `문서작성`, `보고서` 요청이 일반 LLM 답변으로 빠지지 않고 문서작성 스킬 실행으로 라우팅되도록 보강했다.
- 일부 로컬/외부 모델이 출력하는 내부 reasoning 흔적을 최종 응답에서 제거하되, 스트리밍 delta의 공백과 순서는 유지하도록 조정했다.
- 파일찾기 중앙 미리보기 영역을 제거하고, 파일 상세 미리보기를 우측 패널로 이관했다.
- 파일찾기 검색범위와 파일검색 영역을 상하 배치로 정리하고, 인덱스 갱신 결과가 검색범위 영역에 남도록 했다.
- 파일 경로 복사 시 토스트 피드백을 추가하고, 검색 결과 카드 클릭 시 우측 미리보기가 갱신되도록 연결했다.
- 지식폴더 인제스트 진행 중 전체 새로고침 대신 인제스트 작업만 갱신하도록 조정해 엔진 중단 위험을 낮췄다.
- GraphRAG 덤프 미리보기는 중앙 본문에서 제거하고 우측 덤프 뷰어로 이동했다.
- 연결 파일 플로팅 목록에 닫기 버튼을 추가했다.
- 최근 응답 맥락 클릭 시 우측 현재 컨텍스트 상세 영역이 열리도록 연결했다.
